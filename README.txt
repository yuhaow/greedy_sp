These are replicating code for the figures of the paper
Liam Solus, Yuhao Wang, Caroline Uhler. "Consistency Guarantees for Greedy Permutation-Based Causal Inference Algorithms" arXiv:1702.03530
The bash scripts are programmed based on the slurm workflow manager. If you use a different workflow manager, please change the bash scripts to adapt to the workflow manager in your local platform.
For details on how each figure is generated, please refer to the readme files in each folder.
