To get the figures:
First run "data.R" and "sample.R" to get the training data.
Then run the scripts "./autosub_ges.sh", "./autosub_sp_d13.sh", "./autosub_sp_d4.sh", "./autosub_sp_d5.sh", "./autosub_methods.sh" and "./autosub_sp_infinity.sh"
Funally run "tpfig.R" to get the plot for proportion of consistent estimation (Figure 2) and average SHD (Figure 3).
